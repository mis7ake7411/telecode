AI-Powered Telegram Bot

Bridge your Telegram with Claude Code and Codex. Execute commands, process images, transcribe voice, and chat with AI - all from your favorite messenger.

Powerful Features

🤖

Dual AI Engines

Switch between Claude Code and Codex engines on-the-fly. Different chats can use different engines.

💬

Persistent Sessions

Conversations are preserved across messages with intelligent session management.

🖼️

Image Processing

Send photos or screenshots for AI analysis with vision-capable models.

🎤

Voice Messages

Record voice notes that are automatically transcribed via Whisper AI.

🔊

Text-to-Speech

Get audio responses using Fish Audio API for hands-free interaction.

⚙️

Shell Commands

Execute shell commands on your server remotely via the /cli command.

🔒

Access Control

Whitelist users by ID or username to restrict bot access.

⚡

Webhook-Based

Efficient real-time message processing with auto-tunneling support.

🛠️

Developer Friendly

Verbose logging, auto-reload, and comprehensive configuration options.

How It Works

1

Send Message

Send text, images, or voice messages to your Telegram bot

2

Webhook Receives

Telegram delivers your message to the FastAPI webhook server

3

AI Processing

Your chosen engine (Claude or Codex) processes the request

4

Get Response

Receive AI-generated responses instantly in your chat

Quick Start

# Install via pip
pip install telecode

# Run the server
telecode

# The CLI will prompt for your bot token and auto-configure everything
# Get your bot token from @BotFather on Telegram

# Optional: Enable verbose logging
telecode -v

# Optional: Install voice message support
pip install openai-whisper
