# Telecode Landing Page

This directory contains the GitHub Pages landing page for Telecode.

## Viewing the Site

Once GitHub Pages is enabled, the site will be available at:
https://polinom.github.io/telecode/

## Configuration

The `.nojekyll` file prevents Jekyll processing, allowing the plain HTML site to be served directly.

## Local Development

To preview locally, simply open `index.html` in a web browser or use a simple HTTP server:

```bash
cd docs
python3 -m http.server 8080
```

Then visit: http://localhost:8080
